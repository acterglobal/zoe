# Zoe - Personal Workspace App

A modern, joyful Flutter app for organizing thoughts, tasks, and ideas with beautiful simplicity. Built with a Notion-like structure featuring sheets and dynamic content blocks with collaboration capabilities.

## Features

- **Personal Organization**: Create and manage sheets for different aspects of your life
- **8 Content Block Types**: Text, Tasks, Events, Bullet Lists, Links, Documents, Polls, and Lists
- **Real-time Updates**: Instant data persistence with every keystroke
- **Collaboration**: Share sheets with team members, assign tasks, and manage user access
- **Sharing & Deep Linking**: Share sheets and content via links with deep link support for seamless navigation
- **Quick Search**: Powerful search across all content types with filtering capabilities
- **Rich Text Editing**: HTML support with Quill editor for formatted content
- **Media Support**: Upload and manage images, videos, audio files, and documents
- **Task Management**: Assign tasks to team members with due dates and completion tracking
- **Polling System**: Create polls with multiple choice options and vote tracking
- **Profile Management**: Customize user profiles with avatars, bio, and personalization
- **Beautiful UI**: Modern design with smooth animations and intuitive interactions
- **Cross-platform**: Works on iOS, Android, Web, and Desktop
- **Robust State Management**: Built with Riverpod for reliable, reactive state handling

## Architecture

The app follows a clean, modular architecture with separation of concerns:

### Core Components
- **SheetListProvider**: Manages all sheet operations and state using StateNotifier
- **Content System**: Modular content providers for each content type
- **Router Integration**: Go Router for smooth navigation
- **Theme System**: Comprehensive theming with dark/light mode support

### Content Management
- **TextContentListNotifier**: Handles text content with immediate persistence and HTML support
- **TaskListNotifier**: Manages tasks with assignment, due dates, and completion tracking
- **EventsContentListNotifier**: Event scheduling and tracking with start/end dates
- **BulletsContentListNotifier**: Dynamic bullet-point lists with real-time editing
- **LinkListNotifier**: URL link management and organization
- **DocumentListNotifier**: File and media document handling
- **PollListNotifier**: Poll creation, voting, and results tracking
- **ListListNotifier**: Generic list content management

### State Management
- **Riverpod Integration**: Modern reactive state management
- **Provider Family**: Stable controllers for form inputs
- **StateNotifier Pattern**: Predictable state updates
- **Auto-dispose**: Efficient memory management

## Content Block Types

1. **Text Content**: Rich text with HTML support, titles, and descriptions using Quill editor
2. **Tasks**: Task management with checkboxes, descriptions, due dates, and user assignment
3. **Events**: Event scheduling with start/end dates, titles, and detailed descriptions
4. **Bullet Lists**: Dynamic bullet-point lists with real-time editing and descriptions
5. **Links**: URL link management with titles, descriptions, and emoji icons
6. **Documents**: File attachments supporting images, videos, audio, PDFs, and other file types
7. **Polls**: Interactive polls with multiple choice options, voting, and results tracking
8. **Lists**: Generic list containers that can hold various content types
